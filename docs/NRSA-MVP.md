# NRSA LXP — MVP Traceability Matrix (Frontend)

Source of truth is maintained in backend repo at `apex-learn-be/docs/NRSA-MVP.md`.

Frontend MVP focus:
- Auth: email/password + Google OAuth button + callback handler
- Dashboard: programs/courses/modules + progress + points + leaderboard position
- Learning player: module sidebar + lesson types (text/video/audio)
- Assessments: quiz runner + final exam runner (timer/phases/autosubmit)
- Integrity controls: visibility/tab-change + webcam snapshots
- Offline: PWA + cached content + offline progress queue
- Admin: workflow UI + markdown import + analytics views
